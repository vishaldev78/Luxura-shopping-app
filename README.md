# ⚙️ Setup


Clone:

```bash
git clone https://github.com/vishaldev78/Luxura-shopping-app.git
```


Open with Android Studio:

```
Android Studio Ladybug+
JDK 17+
```


Add Firebase:

```
app/google-services.json
```


Run:

```
Build → Run
```
